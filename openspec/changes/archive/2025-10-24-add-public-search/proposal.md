# Add Public Manuscript Search

## Overview
Add search functionality to allow public visitors to search for published manuscripts by title, authors, keywords, and abstract content. This enhances discoverability of published articles on the journal website.

## Motivation
Currently, public users can only browse issues or view individual manuscripts via direct links. There's no way to search for specific content, which limits the usability of the archives. Adding search will improve user experience and make the journal more accessible.

## Scope
- Add search form to the archives page
- Implement backend search endpoint
- Support search by title, authors, keywords, abstract
- Display search results with pagination
- Maintain existing public access restrictions (only published manuscripts)

## Impact
- Improves discoverability of published content
- Enhances user engagement with archives
- No breaking changes to existing functionality
